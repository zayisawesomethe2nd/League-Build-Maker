<p><em>League Build Maker</em> is a website application where players can create in-depth builds for <em>League of Legends</em>. Users can create an account with a username, email, 
and can set their accounts as being premium, allowing them to create public builds that are viewable by others. While creating a build, a user can choose their champion, name their build, 
write a description, select skill path, runes, and item purchases. </p>

</n><p>Remember: If you want to create a public build for other users to see, set your account as being premium! This can also be changed in the settings, along with your password.</p>
